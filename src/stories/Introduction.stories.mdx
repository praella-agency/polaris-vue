# Welcome to Polaris Vue

Polaris Vue by HulkApps is a component library for [Vue.js](https://vuejs.org/) based on
[Shopify Polaris style guide](https://polaris.shopify.com/).
<br />

## Getting Started

Make sure that you had [Vue.js](https://vuejs.org/) **version 2+**.

## Installation
<br/>

#### Using npm or yarn
```bash
npm install @hulkapps/polaris-vue
```

```bash
yarn add @hulkapps/polaris-vue
```

#### Usage for Vue3

```javascript
import { createApp } from 'vue';
import PolarisVue from '@hulkapps/polaris-vue/dist/vue3/polaris-vue.min.js';
import '@hulkapps/polaris-vue/dist/polaris-vue.min.css';

import App from './App';

const app = createApp(App);
app.use(PolarisVue);
app.mount('#app');
```
